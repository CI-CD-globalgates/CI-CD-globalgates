GlobalGates Trade RAG

국가별 수출입 법규 RAG 비서 목업

이 버전은 번역 비서가 아니라 국가별 수출입 법규와 공식 절차를 찾는 비서다. LLM은 질문을 구조화하고 답변을 이해하기 쉽게 설명하며, 실제 근거는 국가별 관세/식품/무역 포털에서 RAG로 찾아온다. MVP 범위는 한국 ↔ 베트남, 일반 통관 + 식품으로 제한한다.

Flow

MVP 처리 흐름

LLM은 검색을 대체하지 않고, 질문을 구조화하고 결과를 설명하는 층으로만 들어간다.

1. Query Intake

자연어 질문 수신. 언어 감지, 질문 정규화, 품목 키워드 추출.

2. Intent Parser

출발국, 도착국, 수출/수입 방향, 품목, 규제 범위, HS code 보유 여부 구조화.

3. Official Retriever

국가/기관/문서종류 메타데이터 필터 후 keyword + vector 하이브리드 검색.

4. Answer Composer

절차, 서류, 검사, 주의사항, 추가 확인 항목으로 구조화해 설명.

5. Guardrail

출처 부족, 국가 불명확, HS code 부재 시 축소 답변과 경고 메시지 추가.

답변 제한 규칙

법규성 정보는 잘 모를 때 더 적게 말해야 한다. 이 섹션은 기능보다 중요하다.

출처가 2개 미만이면

요약 답변만 주고, “추가 확인 필요”를 전면 노출.

품목이 모호하면

HS code 또는 성분표를 요청하고 세부 허가 여부는 단정하지 않음.

국가가 모호하면

출발국/도착국을 다시 묻고 답변을 중단.
