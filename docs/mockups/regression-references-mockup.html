GlobalGates 회귀 모델 — 대기업·플랫폼은
이걸 어떻게 풀고 있나

분류 모델(카테고리 추천)에 이어 다음 작업으로 만들 회귀 모델의 task 후보를 골라보기 위한 사례집이다. 국내·해외 22개 사례를 4개 task 카테고리로 묶어서, 어떤 알고리즘으로 어떤 문제를 풀고 있는지 한눈에 비교한다. 각 카드의 노란 박스는 globalgates에 가져왔을 때의 적용 포인트다.

🎯 globalgates 첫 회귀 모델 — 권장 흐름

국내·해외 사례 22개를 종합하면, 가장 데이터가 풍부하고 분류 모델과 자연스럽게 페어링되는 task는 상품 가격 예측이다. 개발 흐름도 machine-learning 폴더의 c_regression 컨벤션과 자연스럽게 이어진다.

🥇 1순위 — 상품 가격 예측

분류 모델과 페어로 운영

같은 등록 모달에서 카테고리 + 가격을 동시 추천. 학습 데이터는 분류 모델 때 모은 네이버 쇼핑 raw csv의 lprice / hprice 컬럼 그대로 활용. 당근마켓 시세 + Airbnb Smart Pricing + Zillow Zestimate 사례 매칭.

🥈 2순위 — 게시글 인기도 점수

피드 랭킹 알고리즘 출발점

like + reply + bookmark 합산을 회귀 target으로. 시간 가중 회귀 베이스라인부터 GBM까지 단계적 확장. YouTube 시청 시간 + Pinterest engagement + Reddit Hot Score 사례 매칭.

알고리즘 흐름 — c_regression 컨벤션 자연스러운 확장

사용자가 이미 학습한 Linear / Multivariate Linear / Decision Tree 다음 단계로 자연스럽게 이어지는 라인.

1. Linear Regression

베이스라인. 가장 단순, 해석 가능, 빠름.

기 학습됨

2. Ridge / Lasso

정규화로 과적합 방지 + feature 선택 효과.

정규화

3. RandomForest Regressor

비선형 패턴 잡기 + feature 중요도 시각화.

앙상블

4. XGBoost / LightGBM

운영급 성능. Zillow / Airbnb / Booking 표준.
